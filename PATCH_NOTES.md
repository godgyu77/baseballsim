## 게임 밸런스 개선

### 외국인 용병 시스템 개선

**문제점:** 외국인 용병 퀄리티가 너무 낮고 불펜 자원이 많아 연봉 대비 실력이 부족함

**해결 방법:**
- **가격대 비율 조정:**
  - **S급 (Premium):** 연봉 16.0억 이상, 스탯 70~80 (MLB 로테이션급) - **20%**
  - **A급 (Solid):** 연봉 10.0억 이상, 스탯 60~65 (1~2선발급) - **50%**
  - **B급 (Lottery):** 연봉 4.0억 이상, 스탯 50~55 (복권형) - **30%**
- **불펜 자원 제한:**
  - 불펜(RP/CP) 자원은 리스트 25명 중 **0~1명**으로 극히 드물게 생성
  - **95% 이상 선발 투수**로 출력하여 즉시 전력감 확보

**영향:** 외국인 용병의 연봉 대비 실력이 현실적으로 개선되어 전략적 영입이 가능해졌습니다.

### 신인 계약금 현실화

**문제점:** 신인 계약금이 너무 비싸서 구단 운영에 치명적임

**해결 방법:**
- **라운드별 계약금 조정:**
  - **1라운드 (전체 1~10순위):** 1.6억 ~ 5억 원 (최대 5.0억은 초고교급 괴물 신인만)
  - **2라운드:** 1.0억 ~ 1.5억 원
  - **3~5라운드:** 0.3억 ~ 0.9억 원
  - **6~10라운드:** 0.1억 ~ 0.3억 원
- 계약금은 구단 운영에 치명적이지 않은 수준으로 조정

**영향:** 신인 지명 시 재정 부담이 현실적으로 조정되어 탱킹 전략이 가능해졌습니다.

### 재정 시스템 개선

**문제점:** 재정이 그냥 적자라서 무조건적으로 보유자금이 거의 없는 수준으로 변해버림

**해결 방법:**
- **마케팅 및 스폰서십 시스템 추가:**
  - 유니폼/헬멧 스폰서 패치 판매 (A급: +10억, B급: +3억, C급: +5억)
  - 스타 플레이어 굿즈 기획전 (대박: +15억, 평타: +5억)
  - 홈구장 대관 사업 (아이돌 콘서트: +50억, 종교/지역 행사: +10억)
- **시설 및 인프라 투자:**
  - 프리미엄석/스카이박스 리모델링 (50억, 관중 수입 1.2배 영구 증가)
  - 최신식 재활 센터 건립 (80억, 부상 회복 속도 2배, 부상 빈도 30% 감소)
  - 구장 내 식음료 직영점 입점 (30억, 승리 시 +0.5억 추가 수익)
- **자금 집행 원칙:** 모든 투자 및 마케팅 비용은 선택 즉시 보유 자금에서 100% 일시불로 차감

**영향:** 단순한 적자가 아닌 전략적 재정 관리가 가능해져 게임의 깊이가 증가했습니다.

### 헬 모드 재정 현실화

**문제점:** 헬 모드는 1시즌 돌리면 자금이 마이너스가 되버릴 정도로 비현실적임

**해결 방법:**
- **헬 모드 재정 조정:**
  - 수입 80% 감소 (기존 유지)
  - 지출 1.2배 증가 (기존 유지)
  - **하지만 스탯은 공정하게 반영** (스탯 무용지물 방지)
- **마케팅 사업 성공 확률 조정:**
  - 헬 모드에서는 모든 마케팅 사업의 성공 확률이 절반(1/2)으로 감소
  - 대출 이자가 20%에서 **40%**로 폭등

**영향:** 헬 모드가 어렵지만 불합리하지 않게 조정되어 도전적인 난이도로 유지됩니다.

### 난이도별 AI 행동 패턴 구분

**문제점:** "사용자를 어렵고 힘들게 하려고 설계"라는 룰이 모든 난이도에 적용됨

**해결 방법:**
- **헬 모드만 특정 룰 적용:**
  - **헬 모드:** 악덕 에이전트 & 약탈자 (Predatory)
    - 협상: 시장 가치 1.5배(150%) 요구, 1원이라도 깎으면 협상 결렬
    - 트레이드: 동등한 가치 거래 절대 불가, 약탈 로직 적용
    - 용병/FA 경쟁: 플레이어가 필요한 선수에게 +5억 원 더 부르며 방해
    - 성장: 30세부터 조기 에이징, 부상 회복 속도 매우 느림
- **이지/노말 모드:** 일반적인 AI 행동
  - **이지 모드:** 조력자 & 팬보이 (Supporter) - 플레이어에게 유리한 거래 수락
  - **노말 모드:** 비즈니스 파트너 (Rational Agent) - 합리적인 등가교환

**영향:** 난이도에 따라 명확히 구분된 AI 행동 패턴으로 게임의 일관성이 향상되었습니다.

### 스탯 신뢰성 및 가중치 강화

**문제점:** 하드모드 스탯과 타율이 너무 상이함. B+B+A-인 타자인데 OPS가 0.6나옴. 스탯이 무용지물이 됨

**해결 방법:**
- **스탯 신뢰성 보장:**
  - **[불변의 원칙]:** 경기 시뮬레이션 결과는 철저하게 선수의 **20-80 스케일 능력치**를 따름
  - **[클래스 보장]:** 능력치 70 이상(S급)인 선수는 어떤 난이도에서도 리그 상위권 성적을 내야 함
  - **능력치별 성적 하한선:**
    - **60급(A급) 이상 선수:** 아무리 운이 나빠도 50(평균)급 성적 이하로 떨어지지 않게 보정
    - **예시:** 컨택 70인 타자는 헬 모드라도 최소 타율 .280, OPS 0.8은 유지
- **헬 모드 패널티 적용 방식:**
  - **(X) 잘못된 방식:** "헬 모드니까 전체 타율을 -0.05 깎자." -> **절대 금지**
  - **(O) 올바른 방식:** "헬 모드니까 **득점권 찬스(Clutch)**에서만 운을 조금 낮추자." -> 스탯은 예쁘게 찍히지만(타율 .310), 결정적인 타점이 적어 팀은 패배함

**영향:** 스탯이 무용지물이 되지 않고 능력치에 맞는 성적이 보장되어 게임의 신뢰성이 크게 향상되었습니다.

### 2군 기록 보정 시스템

**문제점:** 2군 기록이 너무 낮은 느낌. 1군 기준으로 돌아가는 거 같은 느낌

**해결 방법:**
- **2군 리그 보정 로직 추가:**
  - 2군 경기 시뮬레이션 시, 해당 선수의 모든 스탯에 **가산점(+20)**을 부여한 상태로 결과 산출
  - **공식:** `Simulated_Stat = Actual_Stat + 20 (League Gap)`
- **예시:**
  - 능력치 30(1.5군급)인 타자: 1군 경기 시 타율 .185, 2군 경기 시 보정 적용(30+20=50급) -> 타율 .285
  - 능력치 20(육성선수)인 투수: 1군 경기 시 ERA 22.50, 2군 경기 시 보정 적용(20+20=40급) -> ERA 5.40

**영향:** 2군 선수들이 2군 리그에서 현실적인 성적을 기록하여 게임의 현실성이 향상되었습니다.

### FA 보상 선수 시스템 수정

**문제점:** FA 선수를 데려가면 보상 선수를 FA 선수 데려가는 팀에서 또 데리고 가던 오류 발생

**해결 방법:**
- **보상 선수 이동 로직 명확화:**
  - **절대 원칙:** 보상 선수는 반드시 **"FA 선수를 영입한 구단(Buyer)"**의 로스터에서 **"FA 선수를 뺏긴 원 소속 구단(Seller)"**으로 이동
  - **오류 방지:** 영입한 팀이 보상 선수까지 가져가는 행위는 시스템 오류로 명시
- **등급별 보상 규정:**
  - **S급:** 연봉 300% + 보호선수(20인) 외 1명
  - **A급:** 연봉 200% + 보호선수(20인) 외 1명
  - **B급:** 연봉 100% + 보호선수(25인) 외 1명
  - **C급:** 연봉 150% (보상 선수 없음)

**영향:** FA 보상 시스템이 올바르게 작동하여 게임의 논리성이 향상되었습니다.

### 신인 드래프트 자동 갱신 시스템

**문제점:** 신인 드래프트 자신의 지명 후에 남은 자원들 보여주는 창은 따로 요청해야지 보여줌

**해결 방법:**
- **자동 갱신 원칙 추가:**
  - 사용자가 선수를 지명하면, AI는 즉시 다음 순번 구단들의 지명 결과를 시뮬레이션
  - 그 후, 사용자가 요청하지 않아도 **"지명 결과"**와 함께 **"현재 남은 지명 가능 선수 명단(Remaining List)"을 자동으로 다시 출력**
  - 사용자가 일일이 "남은 선수 보여줘"라고 입력하는 번거로움 제거

**영향:** 신인 드래프트 진행이 더욱 부드럽고 직관적으로 개선되었습니다.

### 선수 포지션 및 구속 표시 개선

**문제점:** 선수들 포지션 및 구속 및 좌투 우투 이상하게 나옴

**해결 방법:**
- **포지션 분류 규칙 명확화:**
  - **내야수 (Infielders):** 포수, 1루수, 2루수, 3루수, 유격수 **만** 해당
  - **외야수 (Outfielders):** 좌익수, 중견수, 우익수 **만** 해당
  - **절대 금지:** 좌익수, 우익수, 중견수를 "내야수"로 표기하거나 분류하는 것을 금지
- **로스터 데이터 정리:**
  - 모든 구단의 타자 로스터를 마크다운 테이블 형식으로 변환
  - 1군과 2군 선수 모두 포지션이 명확히 표시됨

**영향:** 선수 정보의 정확성이 향상되어 게임의 신뢰성이 개선되었습니다.

### 아시아 쿼터 시스템 추가

**신규 기능:** 아시아 쿼터 선수 영입 시스템 추가

**시스템 개요:**
- **보유 한도:** 구단당 **1명** (기존 용병 3명 + 아시아 쿼터 1명 = 총 4명 보유/출전 가능)
- **대상 국적:** 일본(Japan), 대만(Taiwan), 호주(Australia) 국적 소지자
- **연봉 상한선:** 최대 **10.0억 원** (약 70만 달러) 이하
- **스탯 범위:** 종합 능력치 **40(백업/준주전) ~ 60(상위권 주전)** 사이로 생성
- **영입 시기:** 2월 1주차 외국인 용병 영입 기간에 함께 진행
- **동시 출력:** 외국인 용병 명단을 출력할 때, 반드시 그 아래에 아시아 쿼터 후보 명단(15명)도 별도의 표로 함께 출력

**영향:** 외국인 선수 영입의 전략적 깊이가 증가하고, 가성비 좋은 선수 영입 기회가 확대되었습니다.

### 신규 구단 창단 시스템

**신규 기능:** 11번째 구단 창단 기능 추가

**시스템 개요:**
- **특별 지명 드래프트:** 기존 10개 구단의 보호선수(20인) 외에서 총 10명을 지명 (사용자 선택형)
- **외국인 선수 쿼터 확대:** 창단 후 2년간 외국인 선수를 4명까지 보유하고 출전 가능
- **재정 조정:**
  - 특별 지명 보상금(100억)은 초기 자금에서 차감하지 않고 7년 상환 부채로 전환
  - KBO 지원금 50억 원 즉시 지급
  - 구단주 유형별 초기 자금 차등 적용
- **부채 상환:** 매년 12월 연말 정산 시 사용자가 상환액을 선택 (7년 내 상환 완료 필수, 미상환 시 게임오버)

**영향:** 새로운 게임 모드가 추가되어 게임의 재미가 증가했습니다.

## 신규 구단 창단 시스템 개선

### 구단주 유형 선택 시스템

**이전:** 신규 구단 창단 시 구단주 유형이 랜덤으로 배정됨

**현재:** 사용자가 직접 구단주 유형을 선택할 수 있음

#### 구단주 유형 선택 프로세스
- 팀 이름이 결정된 후, 구단주 유형 선택 안내 메시지가 표시됩니다
- 각 유형별 특징, 재정 지원(난이도별), 해고 조건이 명확히 표시됩니다
- 4가지 유형 중 하나를 선택할 수 있습니다:
  - **A유형: 성적 지상주의 (Win-Now)** - 즉각적인 성과를 압박하는 구단주
  - **B유형: 비즈니스맨 (Profit-First)** - 가성비와 흑자 경영을 최우선으로 여기는 구단주
  - **C유형: 시스템/재건 (Rebuilder)** - 장기적인 시스템 구축을 중시하는 구단주
  - **D유형: 의리의 대부 (The Godfather)** - 팀의 로망과 의리를 중시하는 구단주
- 선택한 유형에 따라 초기 자금이 결정되며, 게임 내내 유지됩니다

**영향:** 플레이어가 원하는 운영 스타일에 맞는 구단주를 선택할 수 있어 게임의 재미가 증가했습니다.

## 데이터 일관성 개선

### 보호선수 명단 일관성 보장

**문제점:** 보호선수 20인 외 목록에 포함된 선수가 나중에 보호선수 20인 명단에 나타나는 오류가 발생

**해결 방법:**
- 보호선수 20인 외 목록을 출력하기 전에 각 구단의 보호선수 20인 명단을 먼저 확정하도록 개선
- 보호선수 20인 명단이 확정되면, 그 명단에 포함된 선수는 절대로 "보호선수 20인 외 목록"에 포함되지 않도록 보장
- 사용자가 "보호선수 20인 명단을 보여달라"고 요청할 때, 이전에 출력한 "보호선수 20인 외 목록"과 완벽하게 일치하도록 개선

**적용 범위:** 모든 구단의 보호선수 명단 생성 시 일관성 규칙이 적용됩니다.

## UI/UX 전면 개선

### 제목과 소제목 구분 개선

**문제점:** 제목과 소제목이 구분 없이 똑같아서 눈으로 보기 힘듦

**해결 방법:**
- 마크다운 제목 태그 계층 구조 개선:
  - **H1 (대제목)**: 큰 폰트, 굵은 테두리, 녹색 강조 (`text-3xl`, `font-black`, `border-b-2`)
  - **H2 (중제목)**: 중간 폰트, 얇은 테두리, 진한 녹색 (`text-2xl`, `font-bold`, `border-b`)
  - **H3 (소제목)**: 작은 폰트, 녹색 강조 (`text-xl`, `font-semibold`)
  - **H4 (세부 제목)**: 더 작은 폰트, 회색 (`text-lg`, `font-semibold`)

**영향:** 게임 내 모든 리포트와 선수 목록의 가독성이 크게 향상되었습니다.

### 버튼 배치 최적화

**이전:**
- 상단 헤더: 저장, 불러오기, API 키 설정, 뉴스, 시설관리
- 우측 하단: 지시사항 플로팅 버튼

**현재:**
- 상단 헤더: 저장, 불러오기, API 키 설정만 유지
- 입력창 옆: 지시사항, 뉴스, 시설관리 버튼을 한 줄로 배치

**레이아웃 구조:**

```
[지시사항] [뉴스] [시설관리] | [입력창] [전송]
```

**개선 효과:**

## 게임플레이 변경사항

**이전:** 외국인 용병 퀄리티가 낮고 불펜 자원이 많아 연봉 대비 실력이 부족함

**현재:** 보상 선수는 반드시 영입한 팀에서 뺏긴 팀으로 이동하도록 수정

- **단장 추천 시스템:**
  - 각 은퇴 선수에 대해 단장(플레이어)에게 영구결번 추천 여부를 물어봅니다
  - [OPTIONS] 버튼으로 "영구결번 추천한다/안한다"를 선택할 수 있습니다
  - 추천 기준 안내: 구단 소속 기간, 우승 기여도, 누적 스탯, 상징성, 원클럽맨 여부
- **여론 조사 (단장이 추천한 경우):**
  - 구단주와 팬들의 여론을 시뮬레이션합니다
  - 지지율 계산 요소:
    - 구단 소속 기간 (10년 이상: +20%, 15년 이상: +30%)
    - 우승 기여도 (우승 주역: +25%, 보조 역할: +10%)
    - 누적 스탯 (MVP급: +20%, 올스타급: +10%)
    - 상징성 (프랜차이즈 스타: +15%, 레전드: +20%)
    - 원클럽맨 여부 (원클럽맨: +10%)
- **결과 판정:**
  - **지지율 90% 이상:** 영구결번 확정
  - **지지율 90% 미만:** 일반 은퇴식 진행
  - **단장이 추천 안함:** 일반 은퇴식 진행

**영향:** 단장이 직접 영구결번 후보를 추천하고, 팬과 구단주의 여론을 통해 최종 결정되는 현실적인 시스템이 구현되었습니다.

### 백틱 문자 표시 버그 수정

**문제점:** 마크다운 코드 블록의 백틱(`) 문자가 화면에 그대로 노출되는 버그 발생

**해결 방법:**
- **백틱 문자 제거 로직 추가:**
  - 혼자 떨어진 백틱 1-2개 제거 (3개 이상은 코드 블록으로 보존)
  - 줄 시작/끝의 백틱 제거
  - 공백 사이의 백틱 제거
  - 텍스트 끝의 백틱 제거 (최대 10회 반복)
- 마크다운 코드 블록(```로 시작하는 3개 이상의 백틱)은 유지하고, 혼자 떨어진 백틱만 제거

**영향:** 화면에 불필요한 백틱 문자가 표시되지 않아 가독성이 향상되었습니다.
