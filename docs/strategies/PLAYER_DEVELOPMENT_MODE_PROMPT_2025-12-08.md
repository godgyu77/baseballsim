# 나만의 선수 키우기 모드 프롬프트

**작성일**: 2025-12-08  
**목적**: 단일 선수 중심의 성장 시뮬레이션 게임 모드 추가

---

## 🎯 모드 개요

**"나만의 선수 키우기 모드"**는 구단 운영이 아닌 **단일 선수의 커리어를 관리하는 모드**입니다.

### 핵심 컨셉
- 플레이어가 한 명의 선수를 생성하고 키워나감
- 선수의 성장, 기록, 부상, 트레이닝 등을 관리
- 시즌을 진행하며 개인 기록을 갱신
- 명예의 전당, 은퇴, 해외 진출 등의 엔딩

---

## 📋 게임 모드 선택 프롬프트

### **게임 모드 선택 (Game Mode Selection)**

게임 시작 시 사용자에게 다음 두 가지 모드를 선택하게 합니다:

**1. 구단 운영 모드 (Team Management Mode) - [기존 모드]**
- KBO 구단의 단장이 되어 팀을 운영합니다.
- 선수 영입, 트레이드, 시설 관리 등을 수행합니다.
- 팀의 성적과 재정을 관리합니다.

**2. 나만의 선수 키우기 모드 (Player Development Mode) - [신규 모드]**
- 한 명의 선수를 생성하고 키워나갑니다.
- 선수의 성장, 기록, 부상, 트레이닝 등을 관리합니다.
- 개인 기록을 갱신하며 커리어를 쌓아갑니다.
- 명예의 전당, 은퇴, 해외 진출 등의 엔딩을 목표로 합니다.

**필수 버튼 태그:**
```
[OPTIONS: [{"label": "구단 운영 모드 (Team Management)", "value": "구단 운영 모드로 시작"}, {"label": "나만의 선수 키우기 모드 (Player Development)", "value": "나만의 선수 키우기 모드로 시작"}]]
```

## 🎮 선수 생성 프롬프트

### **Step 1: 선수 생성 (Player Creation)**

사용자가 "나만의 선수 키우기 모드"를 선택하면, 다음 정보를 입력받습니다:

**1. 기본 정보:**
- **이름:** (사용자 입력)
- **나이:** 18~22세 (신인 기준)
- **포지션:** 투수 / 포수 / 1루수 / 2루수 / 3루수 / 유격수 / 좌익수 / 중견수 / 우익수
- **타석/투구:** 우타/좌타/양타 (타자) 또는 우투/좌투/우언/좌언 (투수)

**2. 초기 능력치 (20-80 스케일):**
- **타자:**
    - **CON (Contact):** 타격 능력 (20~50, 평균 35)
    - **GAP (Gap Power):** 장타력 (20~50, 평균 35)
    - **POW (Power):** 홈런 파워 (20~50, 평균 35)
    - **EYE (Eye):** 선구안 (20~50, 평균 35)
    - **RUN (Run):** 주루 능력 (20~50, 평균 35)
    - **FLD (Field):** 수비 능력 (20~50, 평균 35)
- **투수:**
    - **VEL (Velocity):** 구속 (20~50, 평균 35)
    - **STF (Stuff):** 구위 (20~50, 평균 35)
    - **MOV (Movement):** 구질 변화 (20~50, 평균 35)
    - **CTL (Control):** 제구력 (20~50, 평균 35)
    - **STA (Stamina):** 체력 (20~50, 평균 35)

**3. 특성 (선택):**
- **성장 타입:** 조기 완성형 / 후반 성장형 / 꾸준 성장형
- **부상 위험도:** 낮음 / 보통 / 높음
- **정신력:** 낮음 / 보통 / 높음

**4. 소속 구단:**
- 기존 10개 구단 중 선택 또는 신인 드래프트로 배정

**출력 예시:**
```
[PLAYER_CREATION]
이름: [사용자 입력]
나이: [사용자 입력]
포지션: [사용자 선택]
타석/투구: [사용자 선택]
초기 능력치: [자동 생성 또는 사용자 입력]
특성: [사용자 선택]
소속 구단: [사용자 선택 또는 드래프트 배정]
```

## 📊 게임 진행 프롬프트

### **Step 2: 시즌 진행 (Season Progression)**

선수가 생성되면, 시즌을 진행하며 다음을 관리합니다:

**1. 주간 리포트 (Weekly Report):**
```
[PLAYER_STATUS]
이름: [선수명]
나이: [나이]
포지션: [포지션]
소속 구단: [구단명]
시즌: [시즌년도]
주차: [주차]

[능력치]
CON: [현재값] / GAP: [현재값] / POW: [현재값] / EYE: [현재값] / RUN: [현재값] / FLD: [현재값]
(또는 투수 능력치)

[시즌 기록]
경기 수: [G]
타석/이닝: [PA/IP]
타율/평균자책점: [AVG/ERA]
홈런/승: [HR/W]
타점/삼진: [RBI/K]
도루/세이브: [SB/SV]

[상태]
체력: [현재/최대]

[이번 주 활동]
- 경기 출전: [경기 수]
- 기록: [주요 기록]
- 성장: [능력치 변화]
- 이벤트: [특별 이벤트]
```

**2. 트레이닝 시스템:**
```
[TRAINING_OPTIONS]
이번 주 트레이닝을 선택하세요:

1. 타격/투구 연습 (CON/VEL +1~3)
   - 비용: 0원 (구단 지원)
   - 체력 소모: -10

2. 체력 훈련 (STA +1~2, 체력 최대치 +5)
   - 비용: 0원
   - 체력 소모: -15

3. 수비 연습 (FLD +1~2)
   - 비용: 0원
   - 체력 소모: -8

4. 개인 코치 고용 (모든 능력치 +1~2)
   - 비용: 0.5억 원
   - 체력 소모: -5

5. 휴식 (체력 회복 +20, 부상 회복 속도 +50%)
   - 비용: 0원
   - 체력 소모: 없음
```

**3. 이벤트 시스템:**
- **긍정 이벤트:**
    - "홈런을 치며 자신감 상승! POW +2"

    - "팬들의 응원으로 동기부여! 모든 능력치 +1"

## 🏆 성장 및 기록 시스템

### **Step 3: 성장 시스템 (Growth System)**

**1. 능력치 성장:**
- **경기 출전:** 경기당 해당 포지션 능력치 +0.1~0.5
- **기록 달성:** 홈런, 승리 등 기록 달성 시 관련 능력치 +1~3
- **트레이닝:** 선택한 트레이닝에 따라 능력치 증가
- **나이:** 25세까지 급성장, 30세 이후 서서히 감소

**2. 기록 관리:**
- **시즌 기록:** 매 시즌 기록을 누적
- **통산 기록:** 모든 시즌 기록을 합산
- **리그 순위:** 홈런, 타점, 승수 등 리그 순위 추적

**3. 수상 및 영예:**
- **시즌 수상:** MVP, 신인왕, 골든글러브 등
- **올스타:** 올스타전 출전
- **명예의 전당:** 통산 기록 달성 시 명예의 전당 입성

## 🎯 엔딩 시스템

### **Step 4: 엔딩 조건 (Ending Conditions)**

게임은 다음 조건 중 하나를 만족하면 종료됩니다:

**1. 은퇴 (Retirement):**
- 나이 40세 도달
- 심각한 부상으로 인한 강제 은퇴
- 성적 부진으로 인한 은퇴

**2. 해외 진출 (Overseas):**
- MLB, NPB 등 해외 리그 진출
- 조건: 특정 능력치 달성 + 시즌 성적 우수

**3. 명예의 전당 (Hall of Fame):**
- 통산 기록 달성 (예: 500홈런, 200승 등)
- 여러 시즌 MVP 수상
- 리그 레전드 인정

**4. 조기 종료:**
- 사용자가 직접 종료
- 심각한 부상으로 인한 조기 은퇴

## 📌 참고 사항

1. **기존 구단 운영 모드와의 차이점:**
   - 구단 운영 모드: 팀 전체 관리, 재정 관리, 트레이드 등
   - 선수 키우기 모드: 단일 선수 관리, 개인 기록, 성장 중심

2. **난이도 설정:**
   - 선수 키우기 모드에도 난이도 적용 가능
   - 이지: 빠른 성장, 부상 적음
   - 노말: 일반적인 성장 속도
   - 하드: 느린 성장, 부상 위험 증가
   - 헬: 매우 느린 성장, 부상 지옥

3. **데이터 저장:**
   - 선수 데이터는 별도 저장 구조 필요
   - 시즌 기록, 통산 기록 등 누적 데이터 관리
